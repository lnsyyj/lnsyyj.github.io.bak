Python pip安装

```
curl "https://bootstrap.pypa.io/get-pip.py" -o "get-pip.py"
python get-pip.py
```

Python pip更新包

```
pip install --upgrade setuptools
```

Python pip国内源指定

```
pip install setuptools -i https://pypi.tuna.tsinghua.edu.cn/simple
```
