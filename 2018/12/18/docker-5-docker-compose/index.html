2、要创建多个Container

3、要管理这些Container（启动停止删除）

Docker Compose –> 批处理

  • Docker Compose是一个工具

  • 这个工具可以通过一个yml文件定义多容器的docker应用

  • 通过一条命令就可以根据yml文件的定义去创建或者管理这多个容器
