DEMO

CMD

  • 容器启动时默认执行的命令

  • 如果docker run指定了其他命令，CMD命令被忽略

  • 如果定义了多个CMD，只有最后一个会执行

ENTRYPOINT

  • 让容器以应用程序或者服务的形式运行

  • 不会被忽略，一定会执行

  • 最佳实践：写一个shell脚本作为entrypoint
